# 文字接龍遊戲 🎮

一個功能完整的中文文字接龍遊戲，支持單人對戰電腦和雙人模式，並提供完整的詞庫管理系統。

## ✨ 特色功能

### 🎯 遊戲功能
- **單人模式**：與電腦對戰
- **雙人模式**：與朋友輪流出題
- **分數系統**：根據詞語字數計分
- **歷史記錄**：即時顯示所有詞語
- **提示功能**：卡關時提供詞語建議
- **精美界面**：使用 Tkinter 製作的圖形介面

### 📚 詞庫系統
- **豐富詞庫**：目前包含 **560+ 首字**，**1399+ 詞語**
- **自動管理**：使用 NLTK + jieba 中文分詞
- **多種來源**：支持從 Hugging Face 語料庫擴充
- **圖形化管理工具**：方便新增和查詢詞語

## 🚀 快速開始

### 1. 基礎遊戲（無需安裝任何套件）

```bash
python 文字接龍.py
```

### 2. 管理詞庫（圖形化界面）

```bash
python 詞庫管理工具.py
```

### 3. 擴充詞庫（需要 jieba）

```bash
pip install jieba
python build_dict_quick.py
```

## 🎮 遊戲規則

1. 輸入一個 **2-4 個字**的中文詞語
2. 下一個詞必須以**上一個詞的最後一個字開頭**
3. **不能重複**使用已經出現過的詞
4. 每個詞的字數即為得分

### 範例

```
玩家：龍蝦
電腦：蝦子
玩家：子女
電腦：女孩
玩家：孩子
... 以此類推
```

## 🛠️ 詞庫管理

### 方式 1：圖形化管理工具 ⭐ 推薦

最簡單的方式！

```bash
python 詞庫管理工具.py
```

功能：
- ✅ 新增詞語（自動分類）
- 🔍 查詢特定首字的詞語
- 📊 顯示完整詞庫統計
- 💾 匯出統計報告

### 方式 2：快速擴充（需要 jieba）

使用預定義的常用詞組快速擴充：

```bash
pip install jieba
python build_dict_quick.py
```

**注意**：Windows 用戶如果遇到中文檔名問題，請使用 `build_dict_quick.py`（英文檔名）

### 方式 3：進階擴充（Hugging Face + NLTK）

從中文語料庫自動提取詞語：

```bash
pip install -r requirements_詞庫建立.txt
python 建立詞庫_語料庫版.py
```

可選擇的語料庫：
- 中文維基百科
- CLUE 中文語料庫
- 中文新聞語料庫

## 📝 詞庫格式

簡單的文本格式，方便手動編輯：

```
首字|詞語1|詞語2|詞語3|...
```

範例：
```
龍|龍蝦|龍貓|龍舟|龍王
蝦|蝦子|蝦米|蝦仁
貓|貓咪|貓熊|貓頭鷹
```

## 🌟 進階使用

### 自訂詞組

編輯 `build_dict_quick.py`，在 `常用詞` 變數中新增您的詞語：

```python
常用詞 = """
您的詞語 更多詞語 另一個詞
特定領域 專業術語 技術名詞
...
"""
```

### 從文本檔建立詞庫

```python
from build_dict_quick import 簡易詞庫建立器

建立器 = 簡易詞庫建立器()
建立器.從檔案載入("您的文本.txt")
建立器.保存詞庫()
```

## 🐛 常見問題

### Q: Windows 控制台顯示亂碼？

A: 使用 PowerShell 並設置編碼：

```powershell
$env:PYTHONIOENCODING="utf-8"
python build_dict_quick.py
```

或直接使用圖形界面：
```bash
python 文字接龍.py
python 詞庫管理工具.py
```

### Q: 中文檔名無法執行？

A: 使用英文檔名版本：
```bash
python build_dict_quick.py  # 而非 快速建立詞庫.py
```

### Q: 詞庫太大，電腦太強了？

A: 使用詞庫管理工具刪除不需要的詞語，或調整詞庫大小。

### Q: 想要更多詞語？

A: 三種方式：
1. 使用詞庫管理工具手動新增
2. 運行 `build_dict_quick.py` 擴充常用詞
3. 使用 `建立詞庫_語料庫版.py` 從語料庫自動提取
