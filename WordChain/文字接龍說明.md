# 文字接龍遊戲使用說明

## 📁 檔案結構

```
文字接龍.py          # 主程式
詞庫.txt             # 詞語資料庫
詞庫管理工具.py      # 詞庫管理工具（可選）
```

## 🎮 如何執行

```bash
python 文字接龍.py
```

**注意：** 請確保 `詞庫.txt` 和 `文字接龍.py` 在同一個目錄下！

## 📝 詞庫格式說明

詞庫文件 `詞庫.txt` 採用簡單的文字格式：

```
首字|詞語1|詞語2|詞語3|...
```

### 範例

```
龍|龍蝦|龍貓|龍舟|龍王|龍眼|龍捲風
蝦|蝦子|蝦米|蝦仁|蝦餅
貓|貓咪|貓熊|貓頭鷹
```

- 每一行代表以某個字開頭的所有詞語
- 使用 `|` (直線符號) 分隔
- 第一個是開頭的字，後面接該字開頭的所有詞語

## ✏️ 如何擴充詞庫

1. 用文字編輯器開啟 `詞庫.txt`
2. 找到要新增詞語的開頭字那一行
3. 在該行末尾加上 `|新詞語`
4. 如果要新增一個開頭字，就新增一行：`字|詞語1|詞語2`

### 範例：新增詞語

原本：
```
龍|龍蝦|龍貓|龍舟
```

新增後：
```
龍|龍蝦|龍貓|龍舟|龍門|龍袍
```

### 範例：新增開頭字

```
鳳|鳳凰|鳳梨|鳳尾
```

## 🎯 遊戲規則

1. 輸入一個詞語（2-4個字）
2. 下一個詞必須以上一個詞的最後一個字開頭
3. 不能重複使用已經出現過的詞
4. 單人模式：與電腦對戰，看誰先無法接龍
5. 雙人模式：兩位玩家輪流出詞
6. 每個詞的字數即為得分

## 🛠️ 詞庫管理工具

為了更方便地管理詞庫，我們提供了圖形化的詞庫管理工具！

### 執行管理工具

```bash
python 詞庫管理工具.py
```

### 功能介紹

1. **新增詞語**
   - 輸入2-4個字的詞語
   - 系統自動根據首字分類
   - 自動檢查是否重複

2. **查詢詞語**
   - 輸入首字查看所有相關詞語
   - 顯示該字的詞語數量

3. **顯示全部**
   - 查看整個詞庫的內容
   - 按首字排序顯示

4. **匯出統計**
   - 生成詞庫統計報告
   - 顯示詞語數量排行

## 💡 提示

- 遊戲中可以點擊「提示」按鈕獲得詞語建議
- 詞庫越豐富，電腦就越厲害！
- 使用詞庫管理工具可以更方便地擴充詞語
- 可以邀請朋友一起擴充詞庫，讓遊戲更有挑戰性！
