# 詞庫建立工具使用說明

本專案提供三種方式來建立和管理文字接龍遊戲的詞庫。

## 📦 安裝依賴套件

### 基礎遊戲（無需額外安裝）
```bash
# 基礎遊戲只需要 Python 內建的 tkinter
python 文字接龍.py
```

### 快速建立詞庫（只需要 jieba）
```bash
pip install jieba
python 快速建立詞庫.py
```

### 完整語料庫版本（需要完整依賴）
```bash
pip install -r requirements_詞庫建立.txt
python 建立詞庫_語料庫版.py
```

## 🚀 三種詞庫建立方式

### 方式 1：手動管理（圖形化界面）✨ 推薦新手

使用詞庫管理工具，透過圖形界面新增和管理詞語。

```bash
python 詞庫管理工具.py
```

**優點：**
- 圖形化界面，操作簡單
- 可以精確控制每個詞語
- 即時查看統計資訊
- 不需要額外安裝套件

**適合：**
- 想要精確控制詞庫內容
- 逐步擴充詞庫
- 不想安裝額外套件

### 方式 2：快速建立（本地生成）✨ 推薦一般使用

使用預定義的常用詞組快速建立詞庫。

```bash
pip install jieba
python 快速建立詞庫.py
```

**優點：**
- 快速生成（幾秒鐘）
- 只需要 jieba 一個套件
- 包含大量常用中文詞組
- 可以自訂詞組列表

**適合：**
- 想快速開始遊戲
- 不想等待長時間下載
- 網路速度較慢

### 方式 3：語料庫版本（Hugging Face + NLTK）✨ 推薦進階使用

從 Hugging Face 下載中文語料庫，自動提取詞語。

```bash
pip install -r requirements_詞庫建立.txt
python 建立詞庫_語料庫版.py
```

**優點：**
- 詞庫最豐富、最專業
- 使用真實的中文語料
- 可以選擇不同的語料來源
- 自動過濾低頻詞語

**缺點：**
- 需要下載大型數據集（可能需要等待）
- 需要安裝較多依賴套件

**適合：**
- 想要最豐富的詞庫
- 有良好的網路連接
- 願意等待下載和處理時間

## 📚 語料庫來源選項

執行 `建立詞庫_語料庫版.py` 時，可以選擇：

1. **中文維基百科** - 最豐富，但下載較慢
2. **CLUE 中文語料庫** - 學術文本，詞語較正式
3. **中文新聞語料庫** - 時事新聞，詞語較現代
4. **全部** - 整合所有來源（推薦，但需要較長時間）
5. **只使用現有詞庫** - 不下載，只整理現有詞庫

## 🛠️ 詳細使用步驟

### 第一次使用（推薦流程）

#### 步驟 1：快速建立基礎詞庫
```bash
pip install jieba
python 快速建立詞庫.py
```

這會在幾秒鐘內生成包含數千個詞語的詞庫。

#### 步驟 2：測試遊戲
```bash
python 文字接龍.py
```

試玩幾局，看看詞庫是否足夠。

#### 步驟 3：（可選）擴充詞庫

如果覺得詞庫不夠豐富，可以：

**選項 A：使用管理工具手動新增**
```bash
python 詞庫管理工具.py
```

**選項 B：使用語料庫自動擴充**
```bash
pip install -r requirements_詞庫建立.txt
python 建立詞庫_語料庫版.py
# 選擇選項 4（全部）
```

### 進階使用

#### 自訂詞組

編輯 `快速建立詞庫.py` 中的 `常用詞` 變數，新增您想要的詞語：

```python
常用詞 = """
您的詞語 更多詞語 另一個詞
...
"""
```

#### 從自己的文本建立詞庫

修改 `快速建立詞庫.py`，新增：

```python
建立器.從檔案載入("您的文本檔案.txt")
```

## 📊 詞庫格式

詞庫文件採用簡單的文本格式：

```
首字|詞語1|詞語2|詞語3|...
```

例如：
```
龍|龍蝦|龍貓|龍舟|龍王
蝦|蝦子|蝦米|蝦仁
```

## 🔄 更新詞庫

所有建立工具都會自動合併現有的 `詞庫.txt`，不會覆蓋您手動新增的詞語。

## ⚠️ 常見問題

### Q: 下載語料庫很慢怎麼辦？

A: 使用「快速建立詞庫」方式，不需要下載：
```bash
pip install jieba
python 快速建立詞庫.py
```

### Q: 出現 "datasets" 模組錯誤？

A: 這個套件只有語料庫版本需要。如果不使用語料庫版本，不需要安裝。

### Q: 詞庫太大，電腦太強了？

A: 可以在詞庫管理工具中查看並刪除不需要的詞語。

### Q: 想要特定領域的詞語？

A: 使用「快速建立詞庫」，在代碼中新增特定領域的詞組。

## 📝 建議的詞庫大小

- **最小可玩**：500-1000 個詞語
- **一般遊戲**：2000-5000 個詞語
- **豐富體驗**：5000-10000 個詞語
- **專業級別**：10000+ 個詞語
